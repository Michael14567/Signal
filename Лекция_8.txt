Тема: мультипликативная группа поля Галуа, порядки элементов, примитивные элементы и примитивные многочлены



1. Мультипликативная группа поля Галуа

Поле Галуа GF(q) можно рассматривать с двух сторон:

1) как аддитивную группу по операции сложения;
2) как мультипликативную группу по операции умножения.

В мультипликативную группу входят все ненулевые элементы поля.
Поэтому мультипликативная группа поля GF(q) содержит q - 1 элемент.

Если взять ненулевой элемент alpha из GF(q), то его степени

    alpha, alpha^2, alpha^3, ..., alpha^s

также принадлежат GF(q). Так как в поле конечное число ненулевых элементов,
степени alpha рано или поздно начнут повторяться. В частности, существует
такое s, что

    alpha^s = 1.

Минимальное натуральное s, для которого выполняется alpha^s = 1,
называется порядком элемента alpha.

Обозначение:

    ord(alpha) = s.

Порядок элемента делит q - 1:

    s | (q - 1).

Для любого ненулевого элемента поля GF(q) выполняется

    alpha^(q - 1) = 1.

Отсюда для любого элемента поля, включая 0, получается равенство

    x^q - x = 0.

То есть все элементы поля GF(q) являются корнями многочлена

    x^q - x.


2. Утверждение о порядке произведения элементов

Утверждение 1.

Если элементы alpha и beta имеют порядки s_alpha и s_beta, причём эти
порядки взаимно просты, то порядок произведения alpha beta равен произведению
их порядков:

    ord(alpha beta) = s_alpha * s_beta,
    если gcd(s_alpha, s_beta) = 1.

Идея доказательства.

Так как

    alpha^(s_alpha) = 1,
    beta^(s_beta) = 1,

то

    (alpha beta)^(s_alpha s_beta) = 1.

Нужно показать, что меньшая степень не подходит.

Пусть

    (alpha beta)^l = 1.

Тогда можно получить условия, что l должно делиться и на s_alpha,
и на s_beta. Так как s_alpha и s_beta взаимно просты, l должно делиться
на их произведение. Значит, минимальная такая степень равна

    s_alpha s_beta.


3. Существование элементов заданного порядка

Утверждение 2.

Если s^t делит q - 1, то в поле GF(q) существует элемент порядка s^t.

Идея рассуждения.

Пусть

    q - 1 = s^t * u.

Берётся подходящий элемент alpha и рассматривается степень

    beta = alpha^u.

Тогда

    beta^(s^t) = (alpha^u)^(s^t) = alpha^(q - 1) = 1.

Значит, порядок beta является степенью s и делит s^t. При правильном выборе
элемента получается элемент порядка именно s^t.

Следствие.

В поле GF(q) существует элемент порядка q - 1. Такой элемент называется
примитивным элементом поля.

Если

    q - 1 = p_1^(v_1) p_2^(v_2) ... p_s^(v_s),

то для каждого множителя можно найти элемент соответствующего порядка, а затем
перемножить эти элементы. По утверждению о взаимно простых порядках получится
элемент порядка q - 1.


4. Пример: поле GF(7)

В поле GF(7) ненулевые элементы:

    {1, 2, 3, 4, 5, 6}.

Здесь

    q = 7,
    q - 1 = 6.

Рассматриваются степени ненулевых элементов по модулю 7.
Некоторые элементы имеют малый порядок, а некоторые — порядок 6.

Пример:

    3^1 = 3,
    3^2 = 2,
    3^3 = 6,
    3^4 = 4,
    3^5 = 5,
    3^6 = 1   (mod 7).

Значит, элемент 3 имеет порядок 6 и является примитивным элементом GF(7).

Таким образом, мультипликативная группа GF(7)^* циклическая:

    GF(7)^* = {1, 3, 3^2, 3^3, 3^4, 3^5}.


5. Цикличность мультипликативной группы поля Галуа

Мультипликативная группа конечного поля является:

1) абелевой;
2) циклической.

Это означает, что существует такой элемент alpha, что все ненулевые элементы
поля получаются как его степени:

    GF(q)^* = {1, alpha, alpha^2, ..., alpha^(q - 2)},

причём

    alpha^(q - 1) = 1.

Элемент alpha называется образующим, или примитивным, элементом поля.


6. Расширения полей GF(p^m)

Поле GF(p^m) можно построить как расширение поля GF(p) по модулю
неприводимого многочлена степени m.

Идея:

    GF(p^m) = GF(p)[x] / (p(x)),

где p(x) — неприводимый многочлен степени m над GF(p).

В таком поле каждый элемент можно представить многочленом степени меньше m:

    a_0 + a_1 x + ... + a_(m-1) x^(m-1),

где

    a_i ∈ GF(p).

Всего таких элементов:

    p^m.


7. Пример построения GF(2^3)

Рассматривается поле GF(2) и многочлен

    p(x) = 1 + x + x^3.

Так как в фактор-кольце по модулю p(x) выполняется

    x^3 = 1 + x,

можно последовательно находить степени x:

    x^0 = 1,
    x^1 = x,
    x^2 = x^2,
    x^3 = 1 + x,
    x^4 = x + x^2,
    x^5 = 1 + x + x^2,
    x^6 = 1 + x^2,
    x^7 = 1.

Получается 7 ненулевых элементов, то есть x имеет порядок 7.
Следовательно, x является примитивным элементом поля GF(2^3).

Векторное представление элементов через базис {1, x, x^2}:

    1             -> 100
    x             -> 010
    x^2           -> 001
    1 + x         -> 110
    x + x^2       -> 011
    1 + x + x^2   -> 111
    1 + x^2       -> 101


8. Примитивный многочлен

Простой / неприводимый многочлен p(x) степени m с коэффициентами из GF(p)
называется примитивным, если в поле GF(p^m), построенном по модулю p(x),
элемент x является примитивным элементом.

То есть x должен иметь порядок

    p^m - 1.

Иными словами, если p(x) примитивный, то степени x порождают все ненулевые
элементы поля GF(p^m).


9. Характеристика поля

Характеристика поля — это минимальное натуральное число p, для которого

    1 + 1 + ... + 1 = 0
        p раз

Если такого числа нет, говорят, что характеристика равна 0.

Для поля GF(q) характеристика является простым числом p.

В поле характеристики p выполняется:

    p x = 0,

а также тождество

    (x + y)^p = x^p + y^p.

Это связано с тем, что биномиальные коэффициенты C(p, k) при 0 < k < p
делятся на p.


10. Минимальный многочлен

Пусть alpha — элемент расширенного поля GF(p^m). Минимальным многочленом
элемента alpha над GF(p) называется многочлен наименьшей степени с
коэффициентами из GF(p), для которого alpha является корнем:

    m_alpha(alpha) = 0.

Свойства минимального многочлена:

1) m_alpha(x) неприводим над GF(p);
2) любой многочлен f(x) из GF(p)[x], для которого f(alpha) = 0,
   делится на m_alpha(x);
3) m_alpha(x) делит x^(p^m) - x.

Почему любой такой f(x) делится на m_alpha(x):

Пусть

    f(x) = a(x)m_alpha(x) + b(x),

где степень b(x) меньше степени m_alpha(x).

Подставим alpha:

    f(alpha) = a(alpha)m_alpha(alpha) + b(alpha).

Так как f(alpha) = 0 и m_alpha(alpha) = 0, получаем

    b(alpha) = 0.

Но если b(x) ненулевой и имеет степень меньше минимального многочлена,
то это противоречит минимальности m_alpha(x). Значит,

    b(x) = 0,

и f(x) делится на m_alpha(x).


11. Разложение многочлена x^q - x

В поле GF(q) все элементы являются корнями многочлена x^q - x, поэтому

    x^q - x = product_{alpha in GF(q)} (x - alpha).

Для ненулевых элементов также:

    x^(q - 1) - 1 = product_{alpha in GF(q)^*} (x - alpha).


12. Минимальный многочлен и проверочная матрица

В конспекте рассматривается связь минимальных многочленов с кодами.

Если alpha — корень минимального многочлена, то проверочная матрица может
строиться из последовательности степеней alpha:

    H = [1        alpha        alpha^2        ...]
        [1        alpha^2      alpha^4        ...]
        [...]

В двоичном случае элементы GF(2^m) записываются как векторы длины m над GF(2).

Кодовое слово c(x) принадлежит коду тогда и только тогда, когда выполняется
проверочное условие

    c(x) H = 0.

Это эквивалентно тому, что соответствующий многочлен имеет нужные корни,
то есть делится на минимальный многочлен.


13. Пример для GF(2^3)

Берётся примитивный многочлен

    p(x) = 1 + x + x^3.

Пусть alpha — корень p(x). Тогда alpha имеет порядок 7:

    alpha^7 = 1.

Для n = 2^3 - 1 = 7 можно построить код длины 7.
В конспекте указано:

    n = 7,
    k = 4.

То есть рассматривается двоичный циклический код с параметрами вида

    (7, 4).

Элементы поля GF(2^3) записываются через базис:

    1, alpha, alpha^2.

Например:

    1                 -> 100
    alpha             -> 010
    alpha^2           -> 001
    1 + alpha         -> 110
    alpha + alpha^2   -> 011
    1 + alpha + alpha^2 -> 111
    1 + alpha^2       -> 101.

Проверочная матрица строится из этих векторных представлений степеней alpha.


14. Порождающий многочлен циклического кода

Если задан проверочный многочлен h(x), то порождающий многочлен g(x) связан
с ним формулой

    g(x) = (x^n - 1) / h(x).

Для случая

    n = 2^m - 1

получается циклический код, связанный с примитивным элементом поля GF(2^m).


15. Схема кодера

В конце лекции приведена схема кодера на сдвиговом регистре с обратной связью.

Идея схемы:

1) информационные символы m_0, m_1, ..., m_(k-1) поступают в регистр;
2) обратные связи задаются коэффициентами порождающего или проверочного
   многочлена;
3) на выходе формируется кодовое слово циклического кода.

В конспекте указан пример:

    p(x) = 1 + x + x^4,

и переход

    GF(2) -> GF(2^4).

Для такого поля число ненулевых элементов равно

    2^4 - 1 = 15.

Поэтому возникает код длины 15. В записи указан пример параметров

    (15, 4).


Итог лекции

1. Ненулевые элементы поля GF(q) образуют мультипликативную группу из q - 1
   элемента.
2. Порядок элемента alpha — это минимальное s, для которого alpha^s = 1.
3. Порядок любого ненулевого элемента делит q - 1.
4. В поле GF(q) существует примитивный элемент порядка q - 1.
5. Мультипликативная группа конечного поля циклическая.
6. Расширение GF(p^m) строится по неприводимому многочлену степени m.
7. Примитивный многочлен — это неприводимый многочлен, корень которого
   порождает всю мультипликативную группу GF(p^m)^*.
8. Минимальный многочлен элемента alpha — неприводимый многочлен наименьшей
   степени, имеющий alpha корнем.
9. Минимальные и примитивные многочлены используются при построении
   циклических кодов.
10. Порождающий многочлен кода можно выразить через проверочный многочлен:

    g(x) = (x^n - 1) / h(x).

11. Циклические коды можно реализовывать схемами на сдвиговых регистрах
    с обратной связью.
