ОБРАБОТКА И ИНТЕРПЕТАЦИЯ СИГНАЛОВ Лекция 2. Линейные коды и проверочные
матрицы Идейный конспект

  -----------------------------------
  1. Код как множество кодовых слов
  -----------------------------------

Код — это множество кодовых слов.

Эффективность кода определяется: - длиной кодового слова n, -
минимальным расстоянием, - сложностью кодирования и декодирования.

Увеличение длины n обычно увеличивает минимальное расстояние, но
усложняет реализацию.

  -----------------------
  2. Линейный (n,k)-код
  -----------------------

Линейный код — это подпространство пространства GF(q)^n.

Параметры: n — длина кодового слова, k — число информационных символов,
R = k/n — скорость кода.

Размерность подпространства равна k.

  ------------------------
  3. Порождающая матрица
  ------------------------

Порождающая матрица G имеет размер k × n. Строки G — базисные векторы
кодового подпространства.

Кодовое слово: c = mG

где m — информационный вектор длины k.

Все кодовые слова — линейные комбинации строк G.

  ------------------------
  4. Проверочная матрица
  ------------------------

Существует матрица H размера (n−k) × n, такая что для любого кодового
слова:

c H^T = 0

Матрица H задаёт систему линейных ограничений и называется проверочной
матрицей.

Связь: G H^T = 0

  ------------------------
  5. Систематический вид
  ------------------------

Матрицу G можно привести к виду:

G = [ I_k | P ]

где I_k — единичная матрица, P — матрица проверочных коэффициентов.

Тогда

H = [ P^T | I_{n−k} ]

  ---------------------------
  6. Минимальное расстояние
  ---------------------------

Для линейного кода:

d_min = min w(c), c ≠ 0

где w(c) — вес Хэмминга.

Минимальное расстояние равно минимальному числу линейно зависимых
столбцов матрицы H.

Код исправляет t ошибок, если: t ≤ ⌊(d_min − 1)/2⌋

  ----------------------
  7. Граница Синглтона
  ----------------------

Для (n,k)-кода:

d ≤ n − k + 1

Это фундаментальное ограничение на параметры кода.

  -----------------
  8. Дуальный код
  -----------------

Дуальный код C^⊥ имеет: - порождающую матрицу H, - проверочную матрицу
G.

  --------------------------------
  9. Код с проверкой на чётность
  --------------------------------

(n, n−1)-код:

H = (1, 1, …, 1)

Обнаруживает ошибки нечётного веса, но не исправляет одиночные ошибки.

  ----------------------------------
  10. Синдром и исправление ошибок
  ----------------------------------

Пусть принято слово r = c + e.

Тогда:

r H^T = e H^T

Эта величина называется синдромом.

Для одиночной ошибки синдром равен соответствующему столбцу матрицы H.

  ------------------
  11. Код Хэмминга
  ------------------

n = 2^r − 1 k = 2^r − 1 − r d = 3

Исправляет одну ошибку. Является совершенным кодом.

  ------------------------------
  12. Расширенный код Хэмминга
  ------------------------------

(n,k) = (2^r, 2^r − r − 1)

Минимальное расстояние: d = 4

  ------------------------
  Главные идеи лекции №2
  ------------------------

1.  Линейный код — это подпространство.
2.  Код полностью задаётся матрицей G.
3.  Проверочная матрица H задаёт ограничения.
4.  Минимальное расстояние связано со структурой H.
5.  Синдром позволяет определить позицию ошибки.
6.  Код Хэмминга — базовый пример исправляющего кода.
